Массивы
-------

Пустой массив:
```javascript
var array = [];
```

Массив fruits с тремя элементами:

```javascript
var fruits = ["Яблоко", "Апельсин", "Слива"];
```

С массивом как со стеком (конец массива): push / pop

Те же операции с началом массива: shift / unshift

```javascript
console.log("Длина массива: " + arr.length)
```
